# 범주형 데이터 분석 
- 강의 내용

### 적합도 검정

- 관찰도수와 기대도수의 차이
- 빈도(count)로 변경 (관찰값, 기대값)
- scipy.stats.chisquare(observed, expected)
    - observed: 관찰된 빈도 리스트
    - expected: 기대 빈도 리스트

### 독립성 검정

- 두 변수가 서로 독립적인지(연관성이 있는지) 확인
- 교차표 테이블로 만들기
    - 문제에서 표로 주어졌을 때
    - 로우(원) 데이터가 주어졌을 때
- scipy.stats.chi2_contingency(table, correction=True)
    - table: 교차표
    - correction: 연속성 보정 (기본값 True)

### 동질성 검정

- 두 개 이상의 집단에서 동질성을 갖는지 확인
- 검정 절차는 독립성과 같음
